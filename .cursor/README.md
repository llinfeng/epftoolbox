# Cursor Rules

This folder contains Cursor AI coding assistant rules and guidelines for the epftoolbox project.

## Structure

- `rules/main.md` - Main project rules and guidelines covering:
  - Technology stack and dependencies
  - Code style and standards
  - Project structure
  - Development guidelines
  - Common issues and solutions

## Usage

Cursor will automatically read rules from this folder. The rules help the AI assistant:
- Understand project context and structure
- Follow coding standards and conventions
- Maintain compatibility requirements
- Provide accurate suggestions based on project needs

## Adding New Rules

You can add additional rule files in the `rules/` directory. Cursor will read all markdown files in this directory.
